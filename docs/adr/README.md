# Architecture Decision Records

This directory stores Architecture Decision Records (ADRs) for Lineup Lab.

ADRs capture small but important architecture choices:

- the context behind the decision
- the decision itself
- alternatives that were considered
- the consequences for future work

Current ADRs:

- [0001-single-public-api-entrypoint.md](0001-single-public-api-entrypoint.md)
- [0002-auth-owns-users-and-sessions.md](0002-auth-owns-users-and-sessions.md)

Suggested format:

1. Title
2. Status
3. Context
4. Decision
5. Alternatives Considered
6. Consequences
